Predição de Vendas de uma Rede Farmácias

Usando Python, Estatística Descritiva e técnicas supervisionadas de Machine Learning foram realizadas predições das vendas de uma rede de farmácias (Rossmann Stores) usando dados os públicos disponíveis. De acordo com o problema de negócio, foram estimadas as vendas totais para as próximas seis semanas com o objetivo de estimar o investimento que poderia ser feito para reformar toda a rede. Segundo os resultados, prevê-se que, nas próximas seis semanas, a rede de lojas vai realizar, no pior cenário, 6851054 vendas e, no melhor 8610705.

As ferramentas utilizadas foram:

  • Git e Github.
  • Python, PyCharm, Pandas, Matplotlib e Seaborn.
  • Jupyter Notebook.
  • Sklearn, Boruta, XGBoost.
  • Render Cloud.
  • Flask.

Recomendação de Compra e Venda de Imóveis

Usando Python, Estatística Descritiva e Análise Exploratória de Dados foram resolvidas quatro questões de negócio de uma empresa de imóveis chamada House Rocket. Para esse fim foi usado o portfólio da empresa, o qual apresenta diversos atributos para os imóveis. O resultado fundamental mostra que, se a empresa decide seguir as recomendações, o lucro total por revenda pode chegar até 89%, sobre o investimento, se a empresa compra imóveis baratos tanto em boas condições como regulares, assumindo lucros adicionais por reformas.

    As ferramentas utilizadas foram:

  • Anaconda, Pycharm e Jupyter Notebook.
  • Python, Pandas, Numpy e Seaborn.
  • Mapas interativos com Plotly e Folium.
  • Streamlit Cloud.
